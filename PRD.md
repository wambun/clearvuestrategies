## 1. Executive Summary

### Project Overview

## 3. Design Specifications

### Color Palette (Inspired by Framer Template)
| Color | Hex Code | Usage |
|-------|----------|-------|
| Primary Blue | `#4F7AC2` | CTAs, accents, links, highlights |
| Dark Navy | `#001F3F` | Headers, footer background, text |
| Light Background | `#F8FBFE` | Page backgrounds, sections |
| White | `#FFFFFF` | Cards, overlays, light sections |
| Secondary Gray | `#717F98` | Muted text, descriptions |
| Accent Light | `#E8F0FE` | Light accent backgrounds |

### Typography
- **Font Family:** DM Sans (primary), Inter (fallback)
- **Headings:** DM Sans, weights 700-900
- **Body:** DM Sans/Inter, weight 400-500
- **Sizes:**
  - H1: 48-64px (hero)
  - H2: 36-42px (section titles)
  - H3: 24-28px (card titles)
  - Body: 16-18px
  - Small: 14px

### Layout Specifications
- **Desktop Max Width:** 1280px
- **Container Padding:** 60px (desktop), 20px (mobile)
- **Section Spacing:** 100-120px vertical
- **Card Border Radius:** 12-16px
- **Button Border Radius:** 8px

### Animation Patterns
- Scroll-triggered fade-in animations
- Transform: translateY(30px) → translateY(0)
- Opacity: 0 → 1
- Staggered timing for multi-item sequences
- Smooth easing: cubic-bezier(0.44, 0, 0.56, 1)

## 5. Technical Specifications

### Framework & Tools
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn UI (as needed)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Google Fonts (DM Sans)

### Responsive Breakpoints
- **Desktop:** 1280px+
- **Tablet:** 768px - 1279px
- **Mobile:** < 768px

### Performance Requirements
- Lighthouse score > 90
- Lazy loading for images
- Optimized animations (will-change, transform)
- Proper image formats (WebP where possible)

### Accessibility
- Semantic HTML structure
- ARIA labels where needed
- Keyboard navigation
- Color contrast compliance (WCAG AA)
- Alt text on all images
